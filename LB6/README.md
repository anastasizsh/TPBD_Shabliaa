    ```bash
    git clone https://github.com/anastasizsh/TPBD_Shabliaa/LB6.git
2. Перейти в папку проекту:

    ```bash
    cd mongodb_crud
3. Встановити залежності:

    ```bash
    npm install
4. Запустити програму:
    ```bash
    npm start
